# 🖨️ Azamat Print - Полиграфиялық қызметтер орталығы Лендинг

Azamat Print - бұл полиграфиялық қызметтер орталығының құрылған веб-лендинг. Сайт барлық деген ареседе (қайта мобильдік) үлкен ынамдықтың арқасында құрылған.

## 🎯 Бас ойша

- ✅ **Клиенттер**: Қызметтер көру, бағалар, отзывты оқу, байланыс
- ✅ **Қызметкерлер**: Авторизация, қызметтерді өндіктеу
- ✅ **Администраторлар**: Толық басқау (қызметтер, қызметкерлер, байланыс)
- ✅ **Адаптивный дизайн**: Мобильдер үшін оңтайланған
- ✅ **Қазақ тілінде**: Толықтай қазақ тілінде

## 🚀 Іске қосу

### Өндіктеме әмдегі талаптар

- Node.js (v14+)
- npm немесе yarn

### Орнату және іске қосу

```bash
# Репозиториялосын клондау
git clone <repo-url>
cd Azamat-Print-Landing

# Тәуелділіктерін орнату
npm run install-all

# Іске қосу (localhost:3000)
npm start
```

### Өндіктеме

```bash
# Production үшін құру
npm run build

# Vercel-ге orнату
vercel --prod
```

## 📄 Беттер

1. **Басты бет** - Логотип, сипаттамасы, мәзір түймелері
2. **Қызметтер** - Аккордеон форматында барлық қызметтер
3. **Прайс-парақ** - Бағалар, іздеу, сүзгілеу
4. **Пікірлер** - Клиенттердің пікірлері, форма қосу
5. **Байланыс** - Социалды сілтемелер (Instagram, WhatsApp, Telegram, 2GIS)
6. **Қызметкерге кіру** - Авторизация панелі

## 🚢 Орналастыру

### Vercel (Ұсынылған)

```bash
# Vercel CLI орнату
npm i -g vercel

# Логин
vercel login

# Орналастыру
cd Azamat-Print-Landing
vercel --prod
```

### Netlify

```bash
# Build құру
npm run build

# Netlify-ге жүктеу
netlify deploy --prod --dir=client/build
```

### GitHub Pages

1. Repository рәтін құрыңыз
2. Build құрыңыз
3. `gh-pages` ағымын болдыру
